# Evolution Chain Site

This is a deploy-ready Vite + React project.

## Run locally
1. Install Node.js
2. In this folder run:
   npm install
   npm run dev

## Build
npm run build

## Deploy free with GitHub + Vercel
1. Upload this whole folder to a GitHub repo
2. In Vercel, import that repo
3. Click Deploy
4. Use the Vercel URL in Google Sites -> Insert -> Embed
